语法高亮支持

如果在 ``` 后面跟随语言名称，可以有语法高亮的效果哦，比如:

演示 Go 代码高亮
    

     
     
      
    
package main

import "fmt"


func main() {
fmt.Println("Hello, 世界")
}

     
​
演示 Java 高亮
    

     
     
      
    
public class HelloWorld {
    // 1234444
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }

}
