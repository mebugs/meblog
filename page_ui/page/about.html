关于本站

域名与站名

本站是一本个人修炼手册，用于记录个人学习历程、整理问题处理经验、分享工作心得体会、自我充电进步提升！

  • Me：我。
  • Bug：虫、程序漏洞、问题。
  • S：复数。

MeBugS：我的与我所遇到的问题。

域名的意义与本站的核心思想完全符合，这就是本站域名的由来。

至于站名为啥叫米虫？总不能叫【我的问题集】吧。某种意义上的谐音，就叫米虫了。

个人还是比较羡慕米虫这种生物，成为一个米虫，是我人生的终极目标！发一听…

本站构建技术栈

本站是一个独立的个人博客，并没有提供太多社交属性的相关功能。

在早期，米虫折腾过很多开源的建站程序，如：Wordpress、Dedecms、Discuz！、Typecho、PhpWind、Ecshop等。

这其过程中个人从非 IT 行业人士到转行进入 IT 行业，某种意义上转行 IT 也是对 WEB 应用充满了浓烈的兴趣。

在不断折腾下，从一无所知开始自学HTML、CSS、JS到能够快速编写静态网页，基于开源程序使用过或自行修改过不少网站模板，也会扒一些国外的代码研究。

但是无论是多精简和轻便的开源程序始终对于个人而言会存在冗余的功能，在初期访问速度很快，一旦数据量增多之后执行效率就大打折扣。

于是米虫萌生了放弃使用开源建站程序的念头，因此本站并未使用任何开源建站程序。

本站服务端的构建基于 PHP 7.4，数据库使用 MySql 5.7，整体结构为前后端分离形式。

虽然服务端采用了 PHP，但实际所访问的页面都是静态 HTML，并不是简单的伪静态规则或者 Nginx 转发规则。

下图以发布文章举例：

本站前端界面未采用任何UI框架包括 Jquery，JS 交互均采用原生的方式编写，CSS 尽量进行了公共抽离，减少冗余的代码。

  • 本站前端的动画特效均使用 CSS3 特性完成，意味着对于 IE 9 以下的浏览器兼容性不佳。
  • 本站前端实现了自适应，能够有好的支持平台和移动终端的访问。
  • 本站前端采用 PHP 生成的静态 HTML 提供访问，网页关键信息均生成在 HTML 中确保 SEO 效果。

由于所有访问均为静态 HTML，不涉及服务端和数据库交互，这将具备更高的性能和更多的访问压力。

本站管理端引入了Jquery、Vue、Marked 等核心插件，由于管理端不对外开放，在细节性能等方面就相对粗犷。

这里也不再赘述。

关于本站内容

米虫自认为有比较良好的工作学习习惯，在通信期间，米虫整理的 TD-SCDMA 入门到晋阶也给不少同行带来很大的帮助。

转行后，米虫有三四个笔记本已经写满了各种经验/教训/小结，小到 SQL 的语法格式，大到框架融合。

有时候自己寻找查询并不方便，于是这样的个人博客应运而生。

本站主要作用是米虫个人学习整理、经验小结的独立博客。

内容主要涉猎于 IT 行业技术学习总结、问题整理记录、思考分析归纳。

目前主要包含 Java、Go、Python、PHP 四门编程语言。

包含数据库、服务端运维、前端、架构、数据、网络、移动端、分布式的等多个维度。

此外包含个人的一些开源项目、在线案例分享、工具教程和分享等。

关于个人

个人经历概述

米虫，一个九流大专院校毕业的普通学生，所学专业移动通信技术，既不是学生会干部，也不是学霸。

大二那年一家国营通信服务公司到学校搞校企合作顺带招人，老师让我们这些对于未来感到迷茫的小伙伴们也去感受一下。

机缘巧合之下，米虫被选去体验实习两个月，勤恳、积极、主动的工作学习态度受到上级的认可后来被提前录用。

于是米虫放荡不羁的大学生活被提前终止，并走上了通信网络优化这条不归路。

对于比较看重毕业生就业率的大专院校而言，通通绿灯，后续的课程我都没有参与，当然学费还是要正常缴。

由于工作性质原因，接下来的 4 年米虫跑遍了大江南北，在 28 个城市驻留工作，短则两个月，长则一年。

每年除了年会可有可无的在公司总部冒个泡之外，一年 365 天有 360 天在外出差。

这听起来是个很不错的差事，薪水也足够丰厚，但实际上出差工作与外出旅游完全是两个不同的概念。

2014 年，亲人濒临故去，连夜从内蒙古包头做飞机赶往上海，深夜抵达上海市接到撒手人寰的噩耗时，那种撕心裂肺的痛楚令我无法忘怀。

于是个人开始谋求改变，从远途出差申请调动到江浙沪一代，接收降薪降维，工作和未来变得一片迷茫。

2015 年，我申请到家乡工作，被派往邻县，薪水已经折半，工作性质更是降维到无法忍受的地步，我做出了人生中最重要的决定。

跳出舒适圈，重头再来，凭借兴趣选择一条全新的道路，这里也要非常感谢父母的鼓励与支持。

工资再次大幅度缩水，甚至不到原本的四分之一，面对全新的事物，很艰难，但始终有股力量在支撑着我不断前进。

时光如水，恍然优势多年，从萌新到一个九流程序员，说难似乎也没有那么难。

安居一城的米虫不再彷徨，不再东奔西走，爱上一个人，拥有一个家，还有个可爱的闺女，简单的生活，平凡的人生。

此前的种种，偶尔会想起，无论有快乐还是悲伤总是慢慢被藏起，埋在记忆的深处。

无论未来怎么样，我始终坚信：

一切都是最好的选择！

  • 给自己一个目标，正确认知自我，努力拼，才会赢。
